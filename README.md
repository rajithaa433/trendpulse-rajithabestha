# TrendPulse Project

##Overview
This project builds a simple data pipeline that:
1. Collects trending data
2. Cleans the data
3. Analyzes trends
4. Visualizes results

---

##Tasks

### Task 1: Data Collection
- Fetches trending data using pytrends
- Saves data to CSV

### Task 2: Data Processing
- Cleans missing values
- Removes duplicates

### Task 3: Analysis
- Finds most trending topic

### Task 4: Visualization
- Plots trends using matplotlib

---

## Tools Used
- Python
- Pandas
- Matplotlib
- Pytrends
